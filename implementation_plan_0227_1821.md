## 검증 계획

### 자동화 테스트
- `brain/` 디렉토리 내 모든 폴더의 파일 개수를 재조사하여, 개별 폴더당 파일 수가 300개 미만으로 조절되었는지 확인합니다.
- 전체 리소스 사용 용량이 1GB 미만으로 유의미하게 축적되는지 점검합니다.

### 수동 검증
1. 사용자가 에이전트 모드 사이드바를 열었을 때, "Agent Loading" 메시지 없이 모든 대화 리스트가 즉각적으로 표시되는지 확인합니다.
2. "Finalizing Algorithm UAT" 등 로딩이 멈췄던 대화들이 정상적으로 클릭 및 진입 가능한지 확인합니다.
