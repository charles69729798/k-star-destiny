가상 시나리오 설정:

가상의 유저, 스타 정보, 친구 데이터를 생성하여 입력 정보(생년월일시, 성별 등)를 세팅해줘.

입력 및 결과 화면 매칭:

입력값에 따른 출력 화면의 데이터가 사주 로직에 맞게 올바르게 계산되었는지 검토해줘.

4개 국어 정합성 전수 조사:

한글(KO) 분석 결과를 기준으로 영어(EN), 스페인어(ES), 포르투갈어(PT) 번역본이 내용상 왜곡 없이 일치하는지, 텍스트가 화면 밖으로 넘치거나 깨지는 곳은 없는지 점검해줘.

오류 식별:

클릭 후 화면 전환 오류, 데이터 로딩 실패, 언어별 특수문자 깨짐 등 기술적 결함을 리스트업해줘.

리포트 자동 생성 (필수 규칙):

모든 점검 결과를 요약하여 프로젝트 루트에 저장해줘.
