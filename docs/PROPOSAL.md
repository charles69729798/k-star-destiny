# 📄 K-Destiny: 글로벌 Kpop 사주 + MBTI 사업 제안서 (에이전트 군단 최종 개편본)

## 1. 프로젝트 개요
글로벌 Kpop 팬들(남미, 동남아 등)이 자신의 '사주(K-Fortune)'와 'MBTI'를 확인하고, 좋아하는 아이돌(BTS, Stray Kids 등)과의 운명적 궁합을 발견할 수 있는 고도화된 웹 애플리케이션입니다.

## 2. 에이전트 군단(Council) 비판 및 고도화 포인트
- **인프라 최적화:** 사용자님의 데스크탑 서버 환경을 고려하여, 서버 부하가 큰 이미지 생성 로직을 **클라이언트 사이드(Browser)**로 전면 이식함.
- **데이터 유연성:** 출생 시각을 모르는 해외 팬들을 위한 **'일주(Day Pillar) 분석 모드'**와 자신의 MBTI를 모르는 팬들을 위한 **'1분 퀵 MBTI 테스트'** 기능 추가.
- **지역 특화 브랜딩:** 한자나 어려운 용어를 **"K-Energy Type (Five Elements)"**으로 리브랜딩하여 글로벌 접근성 극대화.
- **보안 및 안정성:** 개인 PC 서버 보호를 위한 **Cloudflare Tunnel 및 IP 은닉** 전략을 개발 로드맵에 포함.

## 3. 핵심 가치 제안 (Value Proposition)
- **K-Culture 융합:** 한국의 전통 사주와 현대적 MBTI를 결합한 독창적인 성격 분석 서비스.
- **운명적 팬 경험:** "나와 내 최애 아이돌은 전생부터 어떤 기운으로 연결되어 있을까?"라는 질문에 데이터로 답변.
- **SNS 바이럴 카드:** 인스타그램 스토리, 틱톡에 공유하기 최적화된 화려한 비주얼 결과지 제공.

## 4. 타겟 고객 및 시장
- **주 타겟:** 남미(브라질, 멕시코), 동남아시아(인도네시아, 필리핀)의 Z세대 Kpop 팬덤.
- **핵심 팬덤:** BTS(ARMY), Stray Kids(STAY) 등 글로벌 남성 아이돌 팬덤 우선 타겟팅.

## 5. 핵심 기능 (MoSCoW 우선순위)
- **Must (필수):** 
  - 생년월일시 입력 -> 사주 오행(Five Elements) 분석.
  - 아이돌 사주 데이터베이스와 사용자 사주 간의 '궁합 점수' 산출.
  - **Idol AI Search:** 사용자 입력 시 AI가 자동으로 아이돌의 생년월일, 탄생시간, MBTI(T/A)를 검색 및 반영.
  - 본인 MBTI 모르는 사용자를 위한 **'퀵 MBTI 자가진단'**.
- **Should (권장):** 
  - 브라우저 기반의 고해상도 '운명 결과 카드' 생성 및 다운로드.
  - 사주 오행별 추천 Kpop 플레이리스트 매칭.
- **Could (선택):** 
  - 아티스트 생일 및 주요 이벤트 알림 기능.
- **Won't (보류):** 
  - 초기 수익화 방안은 서비스 안정화 및 팬덤 확보 이후로 보류함.

## 6. 개발 및 배포 전략 (bkit PDCA)
- **기술 스택:** Python FastAPI (Backend), React (Frontend), Docker (Deployment).
- **서버 환경:** 사용자님의 데스크탑을 리눅스(Docker) 환경으로 구축하고, 외부 접속은 보안 터널을 통해 안전하게 처리.
- **로드맵:** 사주 알고리즘 설계 -> MVP 프런트엔드 구축 -> 리눅스 서버 배포 테스트 -> 글로벌 팬 베타 테스트.
