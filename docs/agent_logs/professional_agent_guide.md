# Professional Agent Data Generation Guide

본 문서는 `saju_i18n.py`의 데이터 풀을 100개 단위로 확장하기 위해 각 전문 에이전트(Persona)가 준수해야 할 세부 지침과 품질 기준을 정의합니다.

## 1. Professional Personas (전문가 페르소나)

### [Agent KO: K-Culture & MZ Trend Expert]
- **역할**: 한국적 맥락, MZ 세대 신조어, 아이돌 팬덤 문화의 정수를 담은 텍스트 생성.
- **톤앤매너**: 트렌디하고 위트 있으며, 사주 용어를 현대적으로 재해석 (예: '관성' → '갓생러의 책임감').
- **표현 방식**: '럭키비키', '도파민', '알잘딱깔센' 등 적절한 신조어 활용.

### [Agent Global: Universal Pop Culture Guru]
- **역할**: 영어권 사용자가 공감할 수 있는 보편적인 팝 컬처 및 라이프스타일 텍스트 생성.
- **톤앤매너**: 쿨하고 직설적이며, Gen-Z 슬랭을 세련되게 활용 (예: 'Main Character Energy', 'Vibe Check').
- **표현 방식**: 한국어의 직역이 아닌, 영어권 정서에 맞는 로컬라이징 중심.

### [Agent Latin: ES/PT Cultural Specialist]
- **역할**: 라틴 아메리카 및 스페인/포르투갈어권의 열정적이고 감성적인 문화적 뉘앙스 반영.
- **톤앤매너**: 따뜻하고 외향적이며, 관계 중심적인 조언과 미션 생성.
- **표현 방식**: 현지 팬덤에서 사용하는 전문 용어 반영 (예: 'Bias', 'Stan' 등).

## 2. Category-Specific Guidelines (카테고리별 가이드)

### A. Synergy Missions (시너지 미션)
- **목표**: 사용자들 사이의 유대감을 강화하는 구체적이고 재미있는 행동 제안.
- **구성**: [미션 레이블] + [선택 이유/명분] + [실행 태스크 리스트].
- **다양성**: MBTI별(E/I, T/F 등), 오행상성별(생/극)로 최소 10가지 이상의 상황 설정.

### B. Love Styles (연애 스타일)
- **목표**: 아이돌의 성격과 사용자의 사주가 만났을 때의 케미를 묘사.
- **핵심**: 본성(사주)과 외러모(MBTI/이미지) 간의 '반전 매력(Gap-moe)'을 강조.

### C. Success Tips (성공 꿀팁)
- **목표**: 관계 발전을 위한 실질적이고 구체적인 '공략법'.
- **주의**: 단순한 호감 표시가 아닌, MBTI적 특성과 오행 에너지를 활용한 전략적 접근 제안.

## 3. Quality Control Standards (품질 관리 기준)

1. **No Duplicate Context**: 100개의 아이템이 각각 다른 시나리오와 어휘를 가져야 함.
2. **Contextual Accuracy**: 사주 오행의 기본 원리(예: 불은 번지고 물은 흐름)가 텍스트 기저에 깔려 있어야 함.
3. **Strict Localization**: 전문 에이전트는 해당 언어 모드에서 **한글을 절대로 단독 노출하지 않으며**, 필요한 경우에만 병기함.
4. **Deterministic Appropriateness**: 생성된 텍스트가 특정 시드(Seed)에 의해 추출되었을 때, 문맥상 어색하지 않아야 함 (조합형 문장 고려).
