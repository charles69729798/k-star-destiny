# UAT 시나리오: 복구 기능 및 UX 고도화 통합 검증 (20인)

본 테스트는 가디언 스크립트로 인한 기능 유실 복구 이후, 새롭게 도입된 지능형 아바타 시스템과 MBTI 연동 UX가 실제 사용자 시나리오에서 어떻게 작동하는지 검증하는 것을 목적으로 합니다.

## 👥 UAT 페르소나 그룹 (20인)

### 그룹 A: K-Pop 하드코어 팬 (10인) - 스타 매칭 위주
- **목표**: 좋아하는 아이돌(IU, 정국 등)을 검색하고, 성별에 맞는 아바타가 나오는지, MBTI 궁합 점수가 잘 반영되는지 확인.
- **주요 체크포인트**: 검색 결과의 정확성, 아바타 성별 일치 여부, 추천 아이돌 리스트의 개인화 정도.

### 그룹 B: MZ 커플 및 지인 (5인) - 지인 매칭 위주
- **목표**: 친구나 연인의 정보를 입력(지인 매칭 모드)하고, 사주 분석 결과와 MBTI 궁합을 확인.
- **주요 체크포인트**: 지인 매칭 탭 전환의 매끄러움, 수동 입력 시 성별/MBTI 선택 UI의 편리함.

### 그룹 C: 글로벌 탐험가 (5인) - 다국어 및 비주류 검색
- **목표**: 영어/스페인어 설정에서 비주류 아이돌이나 신인 그룹을 검색.
- **주요 체크포인트**: 다국어 번역 정합성, AI 검색 실패 시 수동 입력 모드로의 부드러운 전환.

## 📝 테스트 시나리오 상세

1. **기능 유실 복구 검증**: 상단 탭을 통해 '스타 매칭'과 '지인 매칭' 모드가 자유롭게 전환되는가?
2. **지능형 아바타 검증**: 고윤정, 카리나(여) 검색 시 여성 아바타가, 차은우, 정국(남) 검색 시 남성 아바타가 나오는가?
3. **MBTI UX 검증**: MBTI 입력 필드 클릭 시 직접 타이핑 대신 팝업 선택창이 뜨며, 선택값이 즉시 반영되는가?
4. **추천 엔진 검증**: 내 MBTI를 입력했을 때 인기 아이돌 섹션의 궁합 점수(%)가 실시간으로 변하는가?

- MBTI 입력 시 오타 발생 가능성 차단 (선택형 UI 도입 효과)
